题目：把一个数组最开始的若干个元素搬到数组的末尾，我们称之为数组的选择。输入一个递增排序的数组的一个旋转，输出旋转数组的最小元素。例如，数组{3,4,5,1,2}为{1,2,3,4,5}的一个旋转，该数组的最小值为1.

```
public class XuanZhuanArr {

/**
 * @Description: 输入一个排序数组的旋转，输出排序数组最小的元素
 * Created by shanzhihong on 2019/07/01-20:28.
 * @version 1.0
 */
    /**
     *  采用二分查找的思想，同时对一些特殊输入做特殊处理
     * @param arr
     * @return
     */
    public int getMinInOrder(int[] arr){
        int low = 0;
        int high = arr.length-1;
        int mid = low;
        //int mid = (low + high)>>>1;
        while (arr[low] >= arr[high]){
            if (high - low == 1){
                mid = high;
                break;
            }else {
                mid = (low + high) >> 1;
                //这里判断特殊情况
                if (arr[low] == arr[mid] && arr[mid] == arr[high]){
                    //只能顺序查询最小的
                    return getMin(arr);
                }
                if (arr[mid] >= arr[low]){
                    low = mid;
                }else if(arr[mid] <= arr[high]){
                    high = mid;
                }
            }
        }
        return arr[mid];
    }
    public int getMin(int[] arr){
        int min = arr[0];
        for (int i=1;i<arr.length;i++){
            if (arr[i] < min){
                min = arr[i];
            }
        }
        return min;
    }
    public static void main(String[] args){

        int[] arr1 = new int[]{4,5,1,2,3};
        int[] arr2 = new int[]{1,0,1,1,1};
        int[] arr3 = new int[]{1,2,3,4,5};
        XuanZhuanArr xuanZhuanArr = new XuanZhuanArr();
        System.out.println("arr1 min "+xuanZhuanArr.getMinInOrder(arr1));
        System.out.println("arr2 min "+xuanZhuanArr.getMinInOrder(arr2));
        System.out.println("arr3 min "+xuanZhuanArr.getMinInOrder(arr3));
    }
}
```
